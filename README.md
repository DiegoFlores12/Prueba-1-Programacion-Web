# Prueba-1-Programacion-Web

Etapa 1: Desarrollo del archivo principal (index.html)

En esta etapa se implementó la estructura base de la página web.

Contenidos principales:
Encabezado con título y descripción del sitio.
Menú de navegación con enlaces internos.
Sección de videojuegos con tres productos.
Sección de trailers con contenido multimedia.
Formulario de contacto.
Características técnicas:
Uso de etiquetas semánticas de HTML5 (header, nav, main, section, article).
Navegación interna mediante anclas.
Integración de contenido multimedia (imágenes, videos y contenido embebido).

Etapa 2: Página de respuesta del formulario (resp-Form.html)

Se desarrolló una segunda página con el objetivo de gestionar la respuesta del formulario de contacto.

Funcionalidad:
Mostrar un mensaje de confirmación de envío.
Informar al usuario que su solicitud ha sido recibida correctamente.
Incluir un enlace que permite regresar al archivo principal (index.html).

Etapa 3: Integración de imágenes

Se creó una carpeta denominada imagenes, en la cual se almacenan los recursos visuales utilizados en la página.

Archivos utilizados:
elden ring.avif
godofwar.avif
deltarune.jpg
Implementación:

Las imágenes fueron integradas en la sección de videojuegos mediante la etiqueta <img>, permitiendo representar visualmente cada producto.

Etapa 4: Integración de videos

Se creó una carpeta denominada videos, destinada al almacenamiento de contenido multimedia local.

Implementación:
Dos videos fueron incorporados mediante la etiqueta <video>, cargados desde la carpeta local.
Un tercer video fue integrado mediante un <iframe> desde la plataforma YouTube.

Esta implementación cumple con el requisito de incluir contenido multimedia tanto local como externo.

Etapa 5: Diseño y estilos (CSS)

Se desarrolló un archivo externo de estilos denominado estilo.css, ubicado en la carpeta css.

Características del diseño:
Uso de una paleta de colores basada en tonos verdes oscuros.
Aplicación de Flexbox para la distribución de elementos.
Diseño de tarjetas para la presentación de videojuegos.
Incorporación de efectos visuales (hover).
Diseño adaptable mediante media queries.
Elementos estilizados:
Encabezado principal.
Menú de navegación.
Secciones de contenido.
Tarjetas de videojuegos.
Contenedores de video.
Formulario de contacto.
